Context: Some camera enthusiast went and described 1,000 cameras based on 13 properties!
    
Content: The 13 properties of each camera  
   (i) Model  
   (i) Release date  
   (i) Max resolution  
   (i) Low resolution  
   (i) Effective pixels  
   (i) Zoom wide (W)  
   (i) Zoom tele (T)  
   (i) Normal focus range  
   (i) Macro focus range  
   (i) Storage included  
   (i) Weight (inc. batteries)  
   (i) Dimensions  
   (i) Price
