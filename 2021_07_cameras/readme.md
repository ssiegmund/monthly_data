Context
    Some camera enthusiast went and described 1,000 cameras based on 13 properties!
    
Content  
   The 13 properties of each camera:  
    Model  
    Release date  
    Max resolution  
    Low resolution  
    Effective pixels  
    Zoom wide (W)  
    Zoom tele (T)  
    Normal focus range  
    Macro focus range  
    Storage included  
    Weight (inc. batteries)  
    Dimensions  
    Price
